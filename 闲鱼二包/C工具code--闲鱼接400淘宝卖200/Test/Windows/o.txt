望江南·超然臺作   
朝代：宋代
作者：蘇軾
原文：
春未老，風細柳斜斜。試上超然臺上看，半壕春水一城花。煙雨暗千家。
寒食後，酒醒卻諮嗟。休對故人思故國，且將新火試新茶。詩酒趁年華。
